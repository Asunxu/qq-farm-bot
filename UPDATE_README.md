# QQ 农场更新日志

记录主要功能更新与问题修复。

## 2026-07-12

### 新增功能

- 设置新增用户级“默认方案”，可统一保存策略设置和自动控制配置。
- 默认方案支持从当前账号导入、恢复系统默认，并可控制新账号是否自动应用。
- 账号管理新增“应用默认方案”，可将当前用户的方案一键覆盖到已有账号并立即同步运行配置。
- 新增代理抓取添加和更新账号，可通过独立抓包服务自动获取 QQ 或微信 Code。
- 后台系统设置新增抓包服务地址、API Token、开放开关和连接测试。
- 代理抓取新增移动端使用说明、证书安装帮助和当前步骤提示。

### Bug 修复

- 修复代理抓取启动账号前未可靠释放手机代理的问题；释放失败会重试并记录错误，避免账号启动后流量继续进入抓包服务。
- 修复 QQ 好友列表分批返回时被过早判断为完整列表的问题，确保好友 GID 收集完成后再释放代理并启动账号。
- 修复手机使用抓包代理时添加结果显示网络错误的问题。
- 阻止旧农场流量重复添加同一账号。

### 优化

- QQ 代理抓取获取 Code 后立即保存账号，完整好友列表同步并释放代理后再启动账号。
- 优化移动端证书打开方式。
- 手机使用手动代理时，后台主机自动直通，抓取期间可持续查看账号添加状态。

## 2026-07-11

### 新增功能

- 活动中心的化肥和有机化肥支持一次兑换多个，并按余额与兑换上限限制数量。
- 登录页图标、标题和提示语支持后台自定义，并全局应用。
- 登录图标支持本地上传，包含动态 SVG；登录页设置可一键恢复默认。
- 新增购买卡密、加入QQ群和更新日志入口，未配置的链接会自动隐藏。

### Bug 修复

- 修复 QQ 会员每日礼包领取。

### 优化

- 运行日志改为按小时保存，仅保留最近三天。

## 2026-07-06

### 新增功能

- 新增四格作物优先种植，可自动规划 `2×2` 土地。
- 自动捣蛋连续失败后暂停到次日，避免重复报错。
- 新账号默认优先使用背包种子，并支持调整或重置种子顺序。

### Bug 修复

- 修复好友农场相关操作。

### 优化

- 优化商城移动端布局、账号切换和网络稳定性。
- 精简运行日志，并优化果实出售结果显示。

## 2026-07-04
