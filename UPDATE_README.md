# QQ 农场更新日志

这里记录最近更新了什么。尽量只写用户能直接感受到的变化。

## 2026-07-12

### 新增功能

- 新增“默认方案”，可以把常用设置保存下来，新账号也能自动套用。
- 账号管理新增“一键应用默认方案”，多个账号不用再一个个改设置。
- 新增“抓包登录”，可按页面提示在手机上添加或更新账号。
- 新增黄金虫功能，支持自动清除自己农场的黄金虫，也可以向好友农场投放。

### Bug 修复

- 修复抓包登录完成后偶尔影响手机网络的问题。
- 修复 QQ 好友没有完全同步就提前结束的问题。
- 修复手机抓包登录时偶尔显示网络错误的问题。
- 修复重复添加同一个账号的问题。

### 优化

- 抓包登录流程更清楚，手机端新增使用说明和证书安装提示。
- 抓包登录添加账号更快，获取到登录信息后会先保存账号。
- 黄金虫投放更稳，会按库存、每日上限和好友静默时间自动控制。
- 活动抽奖完成通知不再挡住右上角账号切换。

## 2026-07-11

### 新增功能

- 活动中心支持一次兑换多个化肥或有机化肥。
- 登录页支持自定义图标、标题和提示语。
- 新增购买卡密、加入 QQ 群和更新日志入口。

### Bug 修复

- 修复 QQ 会员每日礼包领取。

### 优化

- 运行日志只保留最近三天，减少日志占用。

## 2026-07-06

### 新增功能

- 新增四格作物优先种植，会自动安排合适的土地。
- 自动捣蛋连续失败后会暂停到第二天，避免一直报错。
- 新账号默认优先使用背包种子，也可以调整种子顺序。

### Bug 修复

- 修复好友农场相关操作。

### 优化

- 优化商城移动端布局、账号切换和网络稳定性。
- 精简运行日志，果实出售结果显示更清楚。

## 2026-07-04

### 新增功能

- 新增最终阶段施肥策略。
- 购买种子和化肥时可以选择数量，也可以按余额自动计算最多可买多少。
- 放虫放草连续失败后会自动停止。

### 优化

- 金币、点券、金豆豆数量显示更简洁。

## 2026-07-03

### 优化

- 优化手机端页面滚动和顶部栏显示。
