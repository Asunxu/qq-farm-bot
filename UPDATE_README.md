# QQ 农场更新日志

本文档用于记录 `cwser/qq-farm-bot-private` 私有维护版的功能更新、修复和维护记录。

## 2026-07-02

- 修改 UI。
- 修复头像显示。
- 修复经验满后只帮助护主犬的问题。
- 新增“青梅酿万金”活动入口。
- 支持领取青梅种子，并根据抓包补齐领取参数。
- 新增青梅酿酒售卖流程，支持精酿、分享翻倍与售卖。
- 修复青梅酿酒缺少酿造记录导致的 `1034027` 错误。
- 优化活动页移动端 tab 栏，避免“青梅酿万金”在手机上超出屏幕。
- 优化右上角账号切换显示，展示当前账号名称并支持头像字段。
- 去除活动 tab 上不需要的数字徽标。

## 维护说明

- 运行时账号数据、日志、缓存和 `.env` 文件不进入仓库。
- 更新功能时优先参考实际抓包与当前代码结构，避免硬编码账号私有数据。
