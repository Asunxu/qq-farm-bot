# QQ 农场更新日志

记录主要功能更新与问题修复。

## 2026-07-12

- 新增代理抓取添加和更新账号，可通过独立抓包服务自动获取 QQ 或微信 Code。
- QQ 代理抓取支持自动导入好友 GID，并等待好友数据稳定后再完成账号添加。
- 后台系统设置新增抓包服务地址、API Token、开放开关和连接测试。
- 修复手机使用抓包代理时添加结果显示网络错误，并优化移动端证书打开方式。
- 代理抓取新增移动端使用说明、证书安装帮助和当前步骤提示。

## 2026-07-11

- 活动中心的化肥和有机化肥支持一次兑换多个，并按余额与兑换上限限制数量。
- 登录页图标、标题和提示语支持后台自定义，并全局应用。
- 登录图标支持本地上传，包含动态 SVG；登录页设置可一键恢复默认。
- 新增购买卡密、加入QQ群和更新日志入口，未配置的链接会自动隐藏。
- 运行日志改为按小时保存，仅保留最近三天。
- 修复 QQ 会员每日礼包领取。

## 2026-07-06

- 修复好友农场相关操作。
- 新增四格作物优先种植，可自动规划 `2×2` 土地。
- 自动捣蛋连续失败后暂停到次日，避免重复报错。
- 新账号默认优先使用背包种子，并支持调整或重置种子顺序。
- 优化商城移动端布局、账号切换和网络稳定性。
- 精简运行日志，并优化果实出售结果显示。

## 2026-07-04

- 新增最终阶段普通肥和有机肥策略。
- 新增种子、化肥购买数量选择，支持手动输入数量和按余额计算最大可购买数量。
- 放虫放草连续失败后会自动停止。
- 优化金币、点券、金豆豆显示单位：金币超过 1 亿显示为“亿”，点券和金豆豆超过 1 万显示为“万”。

## 2026-07-03

- 优化移动端页面滚动和顶部栏显示。
- 修复管理员弹窗在小屏幕上无法完整查看的问题。

## 2026-07-02

- 优化界面和头像显示。
- 修复经验满级后的好友帮助问题。
- 新增“青梅酿万金”活动及青梅酿酒功能。
- 优化活动页移动端显示。
- 账号切换支持显示账号名称和头像。

## 维护说明

- 运行时账号数据、日志、缓存和 `.env` 文件不进入仓库。
- 更新功能时优先参考实际抓包与当前代码结构，避免硬编码账号私有数据。
